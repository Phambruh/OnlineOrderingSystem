# Online Ordering System

An Android application developed in Android Studio that simulates an online food ordering system with role-based user access.

## Features

* Display menu items using a dynamic user interface
* User and Admin login roles
* CRUD operations for menu management (Create, Read, Update, Delete)
* Local data persistence using Room Database/SQLite
* Shopping cart functionality
* Automatic total price calculation based on selected items
* User-friendly Android interface

## Technologies

* Java
* Android Studio
* Room Database (SQLite)
* RecyclerView
* XML Layouts

This project demonstrates Android application development concepts including database integration, role-based access control, CRUD operations, and e-commerce functionality.
